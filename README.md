## 9. 洛伦兹协变性证明

### 9.1 场方程的四维化

形式完全相同。理论满足狭义相对论要求。

### 9.2 质量定义的张量重构

质量公式(1)是四维动量 \(P^\mu\) 的时间分量。在相对论框架下，应写为协变形式：
\[
P^\mu = \oint_{\Sigma} \frac{\Phi_0 - \Phi(x)}{c^2} \, d^3\Sigma^\mu \tag{48}
\]
其中 \(\Sigma\) 为任意类空超曲面，\(d^3\Sigma^\mu\) 是其法向体积元。当物体运动时，该积分自动给出正确的四维动量变换。

具体地，在物体静止系中，\(\Sigma\) 取等时面，\(d^3\Sigma^0 = dV\)，得 \(P^0 = Mc\)。在运动系中，体积发生洛伦兹收缩，同时 \(\Phi_0 - \Phi\) 按 \(\gamma\) 因子增加（因能量密度增加），两者抵消后 \(P^\mu P_\mu = M^2 c^2\) 不变。

为了验证质量定义的协变性，考虑一个以速度 \(v\) 沿 \(x\) 方向运动的物体。在静止系中，质量公式给出 \(P^0 = Mc\)。在运动系中，物体的体积发生洛伦兹收缩：\(dV' = dV / \gamma\)，其中 \(\gamma = 1/\sqrt{1-v^2/c^2}\)。同时，由于能量密度增加，\(\Phi_0 - \Phi\) 变换为 \(\gamma (\Phi_0 - \Phi)\)（因为 \((\Phi_0 - \Phi)\) 正比于静止能量密度）。因此，运动系中的积分：
\[
